Global Game Jam 2020
03/02/2020

As stressful as it was fun

Yes, it been a while since I've posted on here, but not much has really happened since last time - I've mostly just been working, and I did go back to school in January, but besides that, things have been pretty quiet. Until this past weekend, that is. This weekend was Global Game Jam, and I went, and I'm glad I did.
For those of you that are unfamiliar with the concept of a game jam, basically, you have a certain amount of time to create some type of game (people usually create video games, but you could create board games, for example, if you wanted to). We had 48 hours for this one. We were all put into groups based on which position we signed up for, and I was put into a group of five other people – two other programmers, two artists, and a sound designer.
The theme of the game jam this year was "repair", so after throwing around some ideas, we finally settled on a game that involved a fire-fighting koala (yes, it's as cute as it sounds). We decided to use Unity for the game engine, and so I basically learned C# in one day. After working right up until the deadline, we were finally ready to present our mostly working product. If you're interested, you can check it out here.
